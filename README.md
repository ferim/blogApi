# blogApi
Simple blog api using;
 * .NET Core 6
 * MySql
 * EntityFramework
 * NLog
 * Automapper
 * Swagger
 * Json/XML
 * Onion Architecture
 * Repository Pattern
 * Global Error Handling
 * Async
 * Paging
 * Sorting
 * Filtering
 * Searching
 * Data Shaping
 * HATEOAS
 * Versioning
 * Caching
 * Asp.Net Core Identity
 * Jwt
 * Refresh Token
 * Rate Limiting
 * Throttling
